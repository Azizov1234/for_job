# Drive.net - Premium Car Marketplace & CRM

A modern, SaaS-style web application for an online car marketplace and comprehensive Customer Relationship Management (CRM) built with React and TypeScript.

## 🚀 Technologies

- **React 19 + TypeScript**: Robust frontend framework with strict typing.
- **Vite**: State-of-the-art, lightning-fast bundler.
- **Tailwind CSS v4**: Utility-first CSS framework mapped directly to custom theme styles.
- **Zustand**: Lightweight, fast, and scalable state management.
- **React Router Dom**: Client-side single-page routing (nested layouts).
- **Framer Motion**: Smooth interface transitions and micro-animations.
- **Lucide React**: Clean, modern SVG icon integration.

## ✨ Features

### User Experience
- **Authentication**: Modern Login and Registration interfaces with UX validations and glassmorphic aesthetics.
- **Discovery**: A fully responsive car listing grid populated with luxury vehicles.
- **Purchase Flow**: Smooth, modal-based "Buy Now" flow mimicking real-time API transactions alongside simulated loading and success toast notifications.
- **Micro-interactions**: Hover effects, loading skeletons (pulse indicators), and empty state configurations provide a seamless user journey.

### CRM Dashboard (Admin)
- **Metrics Overview**: Real-time mock dashboard computing total sales volumes, orders processed, and active listings.
- **Inventory Management**: Interface dedicated to tracking, searching, and managing up-to-date luxury vehicle listings.
- **Order Pipeline**: Complete order management data-tables allowing status updates from `PENDING` to `DELIVERED`.

## 📦 Getting Started

### 1. Installation
Clone the repository and install the dependencies:
```bash
git clone https://github.com/Azizov1234/for_job.git
cd for_job_avto_sale
npm install
```

### 2. Development
Run the local setup for development:
```bash
npm run dev
```

### 3. Production Build
Package the application for deployment:
```bash
npm run build
```
